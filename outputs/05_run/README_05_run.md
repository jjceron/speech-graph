# 05_run — Focused Composite SpeechGraph Indices for High/Low Impulsivity Contrast

## 1. Purpose of this run

The purpose of `05_run` was to move from a broad exploratory metric-by-metric screening to a focused and statistically more defensible analysis of SpeechGraph-derived discourse markers. Previous runs showed that individual SpeechGraph metrics produced coherent but weak effects after global multiple-comparison correction. The `05_run` therefore tested whether theoretically interpretable composite indices of graph recurrence and compactness could distinguish subjects labeled as `high_imp` from subjects labeled as `low_imp`.

The guiding research question was:

> Do high-impulsivity subjects show higher SpeechGraph recurrence and compactness indices than low-impulsivity subjects, after adjustment for school year and gender?

This analysis did not redefine the SpeechGraph construction. It reused the graph metrics extracted in `01_run`, where each speech sequence was represented as a directed word-transition graph.

---

## 2. Input data and reused graph construction

### 2.1 Input file

The analysis used the subject/activity/window-level feature table generated by the original SpeechGraph extraction pipeline:

```text
outputs/01_run/analysis/activity_window_features.csv
```

This file already contains one row per valid:

```text
subject × activity × window_size
```

and includes the high/low impulsivity label:

```text
Tipo ∈ {high_imp, low_imp}
```

### 2.2 SpeechGraph construction inherited from `01_run`

The graph definition was kept fixed. For each cleaned token sequence, a directed graph was created from consecutive word transitions:

```text
token_i → token_{i+1}
```

For example, the sequence:

```text
había una vez un pollo
```

produces the directed transitions:

```text
había → una
una → vez
vez → un
un → pollo
```

The graph metrics used across the project included:

```text
nodes, edges, re, pe, l1, l2, l3, lcc, lsc, atd,
density, diameter, asp, cc
```

The `05_run` did not recompute graphs and did not change tokenization, windowing, or graph metric definitions.

---

## 3. Why a composite-index analysis was needed

The preceding exploratory group-contrast analysis (`04_run`) tested many individual SpeechGraph metrics across multiple activities and window sizes. That run revealed a coherent directional pattern: high-impulsivity subjects tended to show greater recurrence, density, and compactness in specific activity-window schemes. However, individual metric-level effects did not survive global FDR correction across the full search space.

The `05_run` was therefore designed as a focused follow-up. Instead of testing hundreds of isolated metrics, it tested three primary composite indices derived from the most coherent patterns observed in `04_run`.

This design reduced the multiple-comparison burden and shifted the analysis from isolated metrics to interpretable graph-structural constructs.

---

## 4. What was used and what was not used

### 4.1 Used in the primary analysis

The primary analysis used:

- SpeechGraph metrics from `activity_window_features.csv`.
- The high/low impulsivity label `Tipo`.
- `School year` and `Gender` as primary covariates.
- Rank-normalized component metrics before constructing composite indices.
- Bootstrap confidence intervals.
- Permutation testing of the group effect.
- FDR correction across the three primary composite indices.

### 4.2 Used only for sensitivity analysis

`Age` was not included in the primary model because it was highly correlated with `School year` in the dataset. Instead, it was used in sensitivity models:

```text
index ~ Tipo + Age + Gender
index ~ Tipo + Age + School year + Gender
```

This allowed the analysis to test whether the group effect was stable under alternative developmental covariate specifications.

### 4.3 Not used as outcomes in this run

This run did not use the continuous Barratt dimensions (`TOTAL`, `NPLAN`, `MOT`, `COG`) as outcomes. The outcome of interest was the SpeechGraph composite index, and the main predictor was high/low impulsivity group.

### 4.4 Not used in the primary model

The analysis did not use:

- Profile assignments from `03_run`.
- Ridge models from `02_run`.
- SHAP values from `02_run`.
- Metadata-only or multimodal clustering outputs.
- All individual SpeechGraph metrics as separate hypothesis tests.

The goal was not prediction or profiling. The goal was to test whether focused SpeechGraph indices differed between high- and low-impulsivity groups.

---

## 5. Composite index definitions

Before index construction, component metrics were transformed using rank-normalization. This reduced the influence of skewness and outliers while preserving the relative ordering of subjects.

Let `z_rank(x)` denote the rank-normalized version of a metric `x`.

### 5.1 A2 recurrence index

The first index captured recurrence in Activity 2 across W10, W20, and W30. It was based on parallel/repeated transitions and two-step cycles:

```text
A2_recurrence_index = mean_z(
    A2_W10_pe,
    A2_W20_pe,
    A2_W30_pe,
    A2_W10_l2,
    A2_W20_l2,
    A2_W30_l2
)
```

Interpretation:

> Higher values indicate more repeated/parallel transitions and more two-step graph cycles in Activity 2.

Expected direction:

```text
high_imp > low_imp
```

### 5.2 A1 W20 compactness index

The second index captured compactness in Activity 1 with window size 20. It combined higher average degree and density with lower graph size and path extent:

```text
A1_W20_compactness_index =
    mean_z(A1_W20_atd, A1_W20_density)
    - mean_z(A1_W20_nodes, A1_W20_asp, A1_W20_diameter)
```

Interpretation:

> Higher values indicate denser, more compact, and less spatially extended word-transition graphs.

Expected direction:

```text
high_imp > low_imp
```

### 5.3 A4 compact-recurrence index

The third index captured a broader compact-recurrence pattern in Activity 4 across W10, W20, and W30:

```text
A4_compact_recurrence_index =
    mean_z(
        A4_W10_re, A4_W20_re, A4_W30_re,
        A4_W10_density, A4_W20_density, A4_W30_density,
        A4_W10_l2, A4_W20_l2, A4_W30_l2
    )
    - mean_z(
        A4_W10_nodes, A4_W20_nodes, A4_W30_nodes,
        A4_W10_asp, A4_W20_asp, A4_W30_asp,
        A4_W10_diameter, A4_W20_diameter, A4_W30_diameter
    )
```

Interpretation:

> Higher values indicate greater recurrence, density, and two-step cycling combined with reduced graph expansion.

Expected direction:

```text
high_imp > low_imp
```

## 6. Statistical model

For each primary composite index, the main model was:

```text
index ~ Tipo + School year + Gender
```

where `Tipo` was encoded as:

```text
high_imp = 1
low_imp  = 0
```

The coefficient of interest was:

```text
β_high = adjusted high_imp − low_imp difference
```

The statistical procedure included:

1. HC3 robust standard errors.
2. One-sided permutation test for the expected direction `high_imp > low_imp`.
3. Bootstrap confidence intervals for the adjusted group coefficient.
4. FDR correction across the three primary indices.
5. Sensitivity models with alternative covariate specifications.

---

## 7. Primary results

All three composite indices showed positive adjusted high-minus-low effects. All bootstrap intervals excluded zero, and all permutation-based FDR-corrected q-values were below 0.01.

| Composite index | n | high_imp | low_imp | Adjusted β | Hedges g | Bootstrap 95% CI | Permutation p | FDR q |
|---|---:|---:|---:|---:|---:|---:|---:|---:|
| `A1_W20_compactness_index` | 185 | 96 | 89 | **0.3546** | 0.3732 | **[0.0808, 0.6167]** | 0.0044 | **0.0088** |
| `A2_recurrence_index` | 218 | 107 | 111 | **0.3269** | 0.3159 | **[0.0626, 0.5870]** | 0.0072 | **0.0088** |
| `A4_compact_recurrence_index` | 252 | 121 | 131 | **0.2777** | 0.2680 | **[0.0449, 0.5062]** | 0.0088 | **0.0088** |

The largest adjusted effect was observed for `A1_W20_compactness_index`, followed by `A2_recurrence_index` and `A4_compact_recurrence_index`.

---

## 8. Sensitivity analysis

Each index was tested under three covariate specifications:

```text
primary_school_year_gender:        index ~ Tipo + School year + Gender
sensitivity_age_gender:            index ~ Tipo + Age + Gender
sensitivity_age_school_year_gender:index ~ Tipo + Age + School year + Gender
```

For all three indices:

- the adjusted high-minus-low beta remained positive;
- the bootstrap confidence interval remained above zero;
- the result was directionally stable across covariate models.

This is important because `Age` and `School year` were strongly correlated in the dataset. The stability of the group coefficient indicates that the SpeechGraph effects were not eliminated by alternative developmental controls.

## 9. Component-level interpretation

The component-level heatmaps showed that the composite-index effects were not driven by a single isolated metric. Instead, the underlying components were directionally coherent.

### 9.1 A2 recurrence index

All Activity 2 recurrence components showed positive high-minus-low effects:

```text
A2_W10_pe
A2_W20_pe
A2_W30_pe
A2_W10_l2
A2_W20_l2
A2_W30_l2
```

This supports the interpretation that high-impulsivity subjects showed more repeated/parallel lexical transitions and more two-step graph cycles in Activity 2.

### 9.2 A1 W20 compactness index

The raw component effects followed the expected compactness pattern:

```text
A1_W20_atd       positive
A1_W20_density   positive
A1_W20_nodes     negative
A1_W20_asp       negative
A1_W20_diameter  negative
```

This indicates that high-impulsivity subjects tended to show denser and more connected graphs, but with fewer nodes and shorter graph extent.

### 9.3 A4 compact-recurrence index

Activity 4 showed a broader compact-recurrence profile:

```text
re, density, l2       positive
nodes, asp, diameter  negative
```

This suggests that high-impulsivity subjects produced graph structures with more recurrence and compactness across several Activity 4 window sizes.

## 10. Relationship among indices

The three composite indices were only weakly to moderately correlated:

| Pair | Correlation |
|---|---:|
| `A1_W20_compactness_index` vs `A2_recurrence_index` | 0.09 |
| `A1_W20_compactness_index` vs `A4_compact_recurrence_index` | 0.20 |
| `A2_recurrence_index` vs `A4_compact_recurrence_index` | 0.27 |

This suggests that the indices capture complementary aspects of discourse graph organization rather than a single redundant factor.

## 12. Main interpretation

The `05_run` provides evidence that high-impulsivity subjects show higher SpeechGraph recurrence and compactness indices than low-impulsivity subjects.

In graph-theoretical terms, high-impulsivity speech was characterized by:

1. More repeated or parallel lexical transitions.
2. More two-step cycles.
3. Higher graph density and average total degree.
4. Reduced graph expansion, as reflected by fewer nodes, shorter average shortest paths, and lower diameter in compactness-related indices.

In linguistic terms, these results suggest that high-impulsivity subjects may produce discourse with more recurrent lexical trajectories and more compact word-transition organization.

---

## 13. Methodological status of the finding

The `05_run` should be interpreted as a focused follow-up analysis derived from the exploratory findings of `04_run`.

This distinction is important:

- The indices were not preregistered before the exploratory analysis.
- The indices were constructed after observing coherent metric-level patterns in `04_run`.
- Therefore, the results should not be described as an independent confirmatory validation.

However, the analysis is still statistically informative because:

1. The number of primary hypotheses was limited to three composite indices.
2. The indices were constructed from interpretable graph-theoretical components.
3. The effects were tested using permutation inference.
4. Bootstrap confidence intervals excluded zero.
5. FDR correction was applied across the primary index family.
6. Effects remained stable under multiple covariate specifications.

A conservative interpretation is therefore:

> A focused composite-index analysis, motivated by prior exploratory screening, found robust evidence that high-impulsivity subjects show greater SpeechGraph recurrence and compactness than low-impulsivity subjects.

---

## 14. Limitations

Several limitations should be considered:

1. The composite indices were derived after exploratory screening and should be externally validated in an independent dataset.
2. The analysis used high/low group labels rather than predicting continuous Barratt scores.
3. The results do not imply that SpeechGraph alone is sufficient for individual-level classification.
4. The graph indices summarize lexical transition structure but do not directly capture semantic content.
5. Activity-specific interpretation depends on the nature of each elicitation task.
6. The analysis does not establish causality between impulsivity and discourse graph structure.

---

## 15. Recommended reporting statement

A suitable summary statement is:

> In a focused follow-up analysis of SpeechGraph composite indices, high-impulsivity subjects showed significantly higher recurrence and compactness of word-transition graphs than low-impulsivity subjects. These effects were observed for Activity 2 recurrence, Activity 1 W20 compactness, and Activity 4 compact-recurrence indices, remained positive across alternative covariate models, and survived FDR correction across the three-index primary family using permutation-based inference.

A more cautious version is:

> Because these indices were derived from prior exploratory screening, the findings should be interpreted as evidence of a promising SpeechGraph marker pattern requiring independent validation.

---

## 16. Conclusion

The `05_run` produced the strongest evidence so far that SpeechGraph analysis captures discourse features related to impulsivity status.

Unlike earlier regression, profiling, and metric-level screening approaches, the composite-index approach revealed a stable and interpretable pattern:

```text
high_imp > low_imp
```

for recurrence and compactness indices derived from word-transition graphs.

The results support the hypothesis that high impulsivity is associated with more recurrent and compact lexical-transition graph organization in speech. This does not yet establish a clinically validated classifier, but it provides a coherent and statistically supported SpeechGraph marker profile for further validation.
