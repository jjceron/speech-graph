# README: Speech-Graph NLP Analysis of Barratt Dimensions

This analysis evaluated whether graph-based features extracted from children and adolescents’ speech transcripts contained predictive information for four Barratt dimensions: TOTAL, NPLAN, MOT, and COG. The pipeline treated the transcript as structured behavioral language data rather than as a bag of words. Only the participant’s speech was retained, each transcript was segmented by activity, transcription labels and non-speech annotations were cleaned, and pause or interruption markers were treated as boundaries so that graph edges were not created across discontinuous speech. For example, within a cleaned activity segment, a sequence such as “había una vez un pollo” is converted into ordered word transitions, where “había → una”, “una → vez”, “vez → un”, and “un → pollo” become directed graph edges. This logic preserves local speech flow and recurrence while avoiding edges that would be artifacts of interviewer turns, pauses, or task transitions.

For each activity, the analysis generated directed speech graphs using sliding windows of 10, 20, and 30 words. With step size 1, a 10-word window is evaluated as words 1–10, then 2–11, then 3–12, and so on. The resulting graph metrics were aggregated at the subject–activity–window level, so the predictive models did not treat every overlapping window as an independent participant. The graph features included measures of lexical quantity and structure such as number of nodes, edges, repeated edges, parallel edges, short-range recurrence metrics, largest connected and strongly connected components, average total degree, density, diameter, average shortest path, and clustering coefficient. Because word count and edge count are nearly fixed within a fixed-size window, the final regression models used 13 non-constant NLP graph features per activity–window scheme.

The graph-derived features were merged with Barratt metadata using subject identifiers. Metadata variables such as age, sex, school, educational level, school year, and group labels were retained for description and stratified interpretation, but they were not used as predictors in the primary NLP-to-Barratt models. This design reduces the risk that the model simply recovers known demographic or derived psychometric group information rather than speech-graph signal. Subjects without metadata were excluded from Barratt modeling because their outcomes could not be evaluated. After the final deduplication correction, the activity-window feature table contained 4,534 rows and no duplicated subject–activity–window observations, which resolved the earlier issue in which some activity-window schemes had more rows than unique subjects.

The predictive analysis fit separate regularized linear models for each combination of Barratt dimension, activity, and window size. This produced 84 primary models: four Barratt targets across seven activities and three window sizes. Each model used cross-validated Ridge regression, with the NLP speech-graph features as predictors and one Barratt dimension as the outcome. This design allowed the analysis to ask whether a specific task and temporal scale of speech organization, for example Activity 7 with a 30-word window, was more informative for a given Barratt dimension than another task or window size.

The main result is that speech-graph NLP features showed weak and highly localized predictive value. The strongest model was for MOT using Activity 7 with a 30-word window, with R² = 0.0171. This indicates that the model explained approximately 1.7% of the out-of-sample variance relative to a mean baseline. The next strongest model was for COG using Activity 3 with a 10-word window, with R² = 0.0123. Additional positive models were smaller: Activity 1 with a 20-word window for MOT, Activity 7 with a 10-word window for COG, Activity 2 with a 20-word window for MOT, Activity 4 with a 30-word window for COG, Activity 2 with a 20-word window for TOTAL, and Activity 4 with a 10-word window for COG. These effects are statistically and practically modest. They should be interpreted as evidence of weak signal rather than as strong prediction.

Across targets, MOT showed the clearest relative evidence of speech-graph signal. Three MOT models had positive R² values with at least 100 subjects, and the best MOT model reached R² = 0.0171. COG also showed limited evidence, with four positive models and best R² = 0.0123. TOTAL showed only one positive model and the effect was negligible, with R² = 0.0018. NPLAN showed no positive models at all; its best model remained negative, indicating that the graph features did not outperform a mean baseline for this dimension.

The error metrics confirm that the positive R² values correspond to very small practical gains. For the best MOT model, RMSE improved from 7.6728 under the mean baseline to 7.6070 under the speech-graph model, and MAE improved from 6.5448 to 6.4189. For the best COG model, RMSE improved from 2.2440 to 2.2302, while MAE improved only from 1.7258 to 1.7218. These differences are directionally favorable but small. They indicate that the NLP features contain some signal, but not enough to support strong individual-level prediction in the present configuration.

The feature-level analysis suggests that the weak predictive signal is mostly related to graph recurrence and connectivity rather than simple lexical volume. For MOT, the features with the largest aggregate contribution across positive models included clustering coefficient, parallel edges, largest strongly connected component, length-3 recurrence, repeated edges, diameter, average shortest path, and length-2 recurrence. In the best MOT model specifically, clustering coefficient was the dominant feature, followed by length-3 recurrence, largest strongly connected component, repeated edges, diameter, average shortest path, length-1 recurrence, and parallel edges. For COG, the most relevant features across positive models were length-1 recurrence, largest strongly connected component, parallel edges, length-3 recurrence, repeated edges, diameter, average shortest path, largest connected component, and number of nodes. These rankings describe model reliance, not causal effects.

The comparison between multivariate model relevance and univariate correlations is important. Some features that contributed strongly to a model also had weak but statistically detectable univariate associations with the Barratt dimension. In the best MOT model, clustering coefficient correlated with MOT at r = 0.1337 and largest strongly connected component at r = 0.1434. In the best COG model, largest strongly connected component, clustering coefficient, and length-3 recurrence showed small positive correlations with COG. Activity 1 with a 20-word window showed stronger univariate correlations for MOT, especially average total degree, density, number of nodes, largest connected component, average shortest path, and diameter, but the multivariate R² remained only 0.0063. This pattern suggests that individual graph features may capture weak associations, but the combined feature set does not yet provide robust predictive improvement. It also suggests substantial redundancy and collinearity among graph metrics.

Several results should be interpreted cautiously. First, many models had negative R² values, meaning that they performed worse than predicting the training mean. Second, 84 activity-window-target models were evaluated, so nominal p-values should be treated as exploratory unless corrected for multiple comparisons. Third, some activity-window schemes had reduced sample sizes because shorter activity responses cannot support larger windows; Activity 5 with a 30-word window had only 37 subjects and should not be considered a primary inferential result. Fourth, the use of overlapping windows is acceptable here because models were fitted after subject-level aggregation, but it still produces highly correlated intermediate measurements. Fifth, the current analysis does not establish clinical utility or causal interpretation; it only evaluates whether graph-based speech structure provides measurable out-of-sample information about Barratt scores.

The most defensible interpretation is that graph-based speech organization contains a weak signal for the motor and cognitive components of impulsivity, with minimal evidence for total Barratt score and no evidence for non-planning impulsivity. The signal appears task- and window-dependent. Activity 7 at longer windows was most informative for MOT, while Activity 3 and Activity 7 at shorter windows were more informative for COG. The relevant features point toward recurrence, connectedness, graph compactness, and local cyclicity as the speech-graph properties most associated with the observed effects.

Future analyses should treat the current results as exploratory and should prioritize validation. Recommended next steps include reporting corrected p-values or false-discovery-rate control for univariate correlations, running permutation tests for the best activity-window-target models, comparing against demographic-only and demographic-plus-NLP baselines, testing more parsimonious feature sets to reduce collinearity, and validating the best models on an independent sample or through repeated nested cross-validation. The current results support continued investigation of speech-graph metrics as behavioral markers, but they do not yet support strong predictive claims for Barratt dimensions at the individual level.
